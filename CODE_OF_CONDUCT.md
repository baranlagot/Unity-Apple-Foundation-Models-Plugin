# Code of Conduct

We are committed to a welcoming, harassment-free project for everyone. Be respectful, assume good intent, give actionable feedback, and focus criticism on the work rather than the person.

Unacceptable behavior includes harassment, discrimination, personal attacks, deliberate intimidation, or publishing another person's private information. Project maintainers may remove content or participation that violates these expectations.

Report conduct concerns privately using the contact path published in the repository's security or community settings.
