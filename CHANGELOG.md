# Changelog

All notable changes to this project are documented here.

## [Unreleased]

### Added

- Initial Unity Package Manager structure and package manifest.
- Provider-based managed API with immutable result and availability models.
- Deterministic editor mock and unsupported-platform providers.
- Streaming, cancellation propagation, structured JSON helper, validation, and tests.
- Initial examples and project documentation.
- Project Settings UI with persisted mock, native logging, timeout, and fallback policy defaults.
- Immutable runtime configuration with settings-aware Editor provider selection.
- Thread-safe native request lifecycle with callback routing, cancellation, timeouts, and late-event cleanup.
- Shared Swift Foundation Models core, iOS C ABI, and idempotent Xcode postprocessor.
- Importable sample scenes for availability, text, streaming/cancellation, and JSON quest generation.
