# Contributing

Thank you for helping improve Apple Foundation Models for Unity.

1. Open an issue before large API or native-bridge changes.
2. Keep platform integration behind `IAppleFoundationModelsProvider`.
3. Add or update tests for changed behavior.
4. Run Unity Edit Mode tests in the minimum supported Unity version.
5. Update documentation and `CHANGELOG.md` when public behavior changes.

Do not commit credentials, provisioning profiles, generated Xcode projects, or proprietary SDK files. By contributing, you agree that your contribution is licensed under the MIT License.
